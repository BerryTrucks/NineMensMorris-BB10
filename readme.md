NineMensMorris: [Muhle] for BlackBerry 10
===========================

NineMensMorris is a Muhle Port for BlackBerry 10 by [summeli]

QMühLe is an implementation of the strategy board game "Nine Men’s Morris" (German: "Mühle") for the MeeGo 1.2 Harmattan platform (Nokia N9, N950), using QML and Qt Components. It supports playing against a human opponent or against the computer with adjustable strength.
